# adapters/max

Стаб MAX-адаптера. Реализация — в Блоке 2б, после регистрации ИП и публикации бота в МАКС.

Цель этой папки сейчас — зафиксировать, что доменное ядро (`core/dialog`) уже работает в режиме
«мессенджер-независимый» и Блок 2б сводится к написанию второго адаптера без правок ядра.

Любой вызов методов адаптера сейчас бросает `Error('MAX adapter not implemented yet, see Block 2б')`.
